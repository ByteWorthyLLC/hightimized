2. Create `lefthook.yml` at the repo root with EXACTLY this content (matches RESEARCH.md "Pattern 4" verbatim — note: vitest stanza is included so when Plan 03 wires vitest, the hook just works):

```yaml
pre-commit:
  parallel: true
  commands:
    typecheck:
      glob: '*.{ts,tsx}'
      run: pnpm tsc --noEmit
    eslint:
      glob: '*.{ts,tsx}'
      run: pnpm eslint {staged_files}
    prettier:
      glob: '*.{ts,tsx,json,md,yaml}'
      run: pnpm prettier --write {staged_files} && git add {staged_files}
    vitest:
      glob: '*.{test,spec}.{ts,tsx}'
      run: pnpm vitest run --reporter=verbose {staged_files}

pre-push:
  commands:
    test-full:
      run: pnpm test:run
```

3. Add `postinstall` to package.json scripts (merge into the existing scripts block from Plan 01):
```json
"postinstall": "lefthook install"
```
