  - skip + `Dear Billing Department,`

  - For each flagged line, draw 3 paragraphs (RESEARCH.md Q4 lines 404-415) using the line's cptCode, description, charged, publishedRate, multiplier values formatted with `$` and `.toFixed(2)`.
